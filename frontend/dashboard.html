Dashboard de Risco de Evasão Escolar

Bem-vindo ao painel. Aqui futuramente você verá visualizações como:

  • Mapa de escolas com risco crítico de evasão
  • Fatores socioeconômicos mais influentes
  • Simuladores de cenários por região

Este projeto utiliza PySpark, Machine Learning e dados do MEC e IBGE para prever escolas com alto risco de evasão escolar.
